# 项目说明文档
- webpack2/ 项目根目录
    + README.md 项目说明文档
    + package.json 依赖包和项目信息的配置文件
    + public/ 静态资源目录
        + index.html 静态页面
    + src/ 项目源码目录
        + js/ JS脚本目录
            + module1.js 模块1
            + module2.js 模块2
        + index.js 入口文件
    + dist/ 打包输出目录
- 注意: 项目根目录名称不能叫 webpack 

- 技术栈
+ Bootstrap
+ 原生JS
+ axios
+ webpack
